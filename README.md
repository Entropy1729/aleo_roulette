# Aleo Roulette

Roulette web app with ZKP.

## Up & Running

To use the project you need to have [nix](https://nix.dev/tutorials/install-nix) installed.

To enter the nix-shell, type:

`nix-shell`

If it's the first time using this app run:

`make init`

To build the project:

`make build`

to run the Frontend:

`make run_front`
